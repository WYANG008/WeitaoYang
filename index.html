Undergraduate Research

Surface plasmon polariton generation using acousto-optic effect in fiber

Surface plasmon polariton (SPP) effect on all-fiber acousto-optic tunable filter is studied. Acoustic wave is used to induce core mode to cladding mode coupling and eventually results in SPP generation at the fiber cladding surface. The interaction between optical fiber core mode, cladding mode and SPP mode is formulated and simulated by mode coupling theory. Experimental studies were carried out to verify the theory and simulation results. Experiential study on SPP generation mechanism using acousto-optic effect in fiber shows TE mode coupling is suppressed but core mode to cladding TM mode enhanced with a resonant wavelength red shift, which agree with the theoretical prediction and simulation result.
